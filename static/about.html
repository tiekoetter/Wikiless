About

Wikiless is a free, open-source Wikipedia alternative, focused on privacy. The project aims to deliver a more secure, anonymous browsing experience by minimizing data collection and tracking, ensuring users can explore knowledge with enhanced privacy.

The source is available at Github https://github.com/tiekoetter/Wikiless.


  • Due to high bot traffic this instance REQUIRES JavaScript to be enabled. This instance uses Anubis as a bot protection gateway. Normally no JavaScript is required.
  • No ads.
  • All requests go through the backend, client never talks to Wikipedia.
  • Prevents Wikipedia getting your IP address.
  • Self-hostable. Anyone can setup a private or public instance.
