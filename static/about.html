About

Wikiless is a free open source alternative Wikipedia front-end focused on privacy.

The source is available at Github https://github.com/tiekoetter/wikiless.


The original developer that made Wikiless is, "Orenom". He has gotten DMCAed' and "Metastem" has forked it to continue keep it maintained.

This is a fork of https://github.com/Metastem/Wikiless.

  • No Javascript or ads.
  • All requests go through the backend, client never talks to Wikipedia.
  • Prevents Wikipedia getting your IP address.
  • Self-hostable. Anyone can setup a private or public instance.
